FCP assessment.

Author @Dan Asafo-Agyei

Run in command line with "python simulator.py"

Each entered percentage is the probability of each person either:

Infecting another person
Recovering and becoming immune
Dying from infection

Click each button to save valid input

Entered values should not exceed > 100%

Logs of each day of simulation stored in folder simulation_logs when graph window closed.

Note: Entering > 10,000 people causes program to run slow
